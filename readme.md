Fizzbuzz
==================

###Creating a simple TDD Fizzbuzz Kata in CoffeeScript

####Technologies Used:

  - CoffeeScript
  - Mocha
  - Chai
  - Node

How to use
----------
Clone the repository:
```shell
$ git clone git@github.com:scully87/Coffee_Buzz.git
```

Change into the directory:
```shell
$ cd coffee_buzz
```

Install the package:
```shell
$ npm install
```

Run the tests:
```shell
  mocha --compilers coffee:coffee-script
```
